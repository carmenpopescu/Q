Class ClassConfig

Description
Description | Vars (details) | Methods (details)

Implements interfaces:

  • ArrayAccess (internal interface)

Hold static (config) properties for a class, wich may set it's configuration when it is loaded.

This prevents loading classes that are not used.

Use ClassConfig::forClass(classname) to create a bin. Don't use virtual properties or the virtual return of a call as property or argument. This will result in unexpected behaviour.
