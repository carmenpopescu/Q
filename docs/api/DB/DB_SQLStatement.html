addCriteria (line 409)

Add criteria as where or having statement as $column=$value.

If $value == null and $compare == '=', $compare becomes 'IS NULL'.

NOTE: This function does not escape $column

  • access: public
DB_SQLStatement addCriteria (mixed $column, mixed $value, [string $compare = "="], [int $options = 0], [int $subset = 0])
  • mixed $column: Column name, column number or array of column names ($column[0]=$value OR $column[1]=$value)
  • mixed $value: Value or array of values ($column=$value[0] OR $column=$value[1])
  • string $compare: Comparision operator: =, !=, >, <, =>, <=, LIKE, LIKE%, %LIKE%, REVERSE LIKE (value LIKE column), IN, NOT IN, ALL and BETWEEN
  • int $options: Addition options (language specific) as binairy set
  • int $subset: Specify to wich subset the change applies (0=main query)
